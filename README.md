# Math Calc - mc
Uhh... a minimalistic calculator i guess

i wanted a calculator that can handle factorials, combinations,
permutations, etc.

## Syntax
 - Normal
 - Powers `n^k`
 - Factorials `n!`
 - Combinations `nCk`
 - Permutations `nPk` - not yet
